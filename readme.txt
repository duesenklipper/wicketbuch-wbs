Die WBS-Anwendung kann in drei unterschiedlichen Rollen verwendet werden:

	* Benutzer
	* Projekt-Administrator (Manager)
	* Administrator

Diese Rollen habe dabei folgende Funktionsmöglichkeiten:

1. Benutzer

	* Eintragen / Ändern von der Tasks
	* Anpassen der eigenen Einstellungen

2. Projekt-Administrator

	* Ansicht der Projektstatistiken
	* Google-Maps
	* Eintragen / Ändern von der Tasks
	* Anpassen der eigenen Einstellungen

3. Administrator

	* Anlegen / Ändern von Benutzern
	* JMX-Konfiguration
	* Ansicht der Projektstatistiken
	* Google-Maps
	* Eintragen / Ändern von der Tasks
	* Anpassen der eigenen Einstellungen
